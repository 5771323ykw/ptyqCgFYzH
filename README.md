# 图书管理系统简介 java650

## 项目概述

本系统是基于 Vue3 和 Node.js 开发的图书管理系统，旨在提供便捷的图书信息管理功能。

## 技术栈

- 前端：Vue3
- 后端：Node.js
- 数据库：MySQL
- 开发工具：VSCode
- Node.js版本：v18.17.1

## 功能模块

### 系统角色

- 管理员

### 主要功能

- 图书录入
- 图书查询
- 图书修改
- 图书删除

## 系统运行环境

- 前端：现代浏览器
- 后端：Node.js v18.17.1 环境下运行
- 数据库：MySQL

## 目录结构

```
 project/
 ├── public/             # 公共文件目录
 ├── src/                # 源代码目录
 │   ├── assets/         # 资源文件
 │   ├── components/     # Vue 组件
 │   ├── views/          # 页面文件
 │   ├── router/         # 路由配置
 │   ├── store/          # 状态管理
 │   ├── App.vue         # 主组件
 │   └── main.js         # 入口文件
 ├── package.json        # 项目描述文件
 └── ...
```

## 部署步骤

1. 准备运行环境，安装 Node.js 和 MySQL。
2. 初始化数据库，导入数据库文件。
3. 安装项目依赖，运行后端服务。
4. 前端代码打包部署。

## 核心亮点

- 基于 Node.js，提供高效的后端服务。
- 系统界面简洁，操作便捷。
